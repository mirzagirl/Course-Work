Property Definition Example Usage Visual Example
Transition Applies smooth transitions when CSS properties change states (e.g., on hover). transition: transform 0.5s ease;
:hover { transform: rotate(45deg); }
(Hover the box)
Animation Executes predefined keyframe animations automatically or triggered by events. animation: spin 2s linear infinite;
@keyframes spin { from { transform: rotate(0deg); } to { transform: rotate(360deg); } }
(Continuous animation)
